

**Change directory to /path**
```sh
admin  Lynn shell-cheatsheet ∫ cd /path
```
***Shortcut***
- Home directory
```sh
admin  Lynn ∫ cd
```

- Previous directory
```sh
admin  Lynn ∫ cd -
```

- Home directory of user_name
```sh
admin  Lynn ∫ cd ~user_name
```
